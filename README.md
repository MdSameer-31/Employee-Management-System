# Employee Management System (Java + Swing + MySQL)

A simple desktop-based application built using **Java Swing** and **MySQL** to manage employee records.  
Implements **CRUD operations** with a user-friendly GUI and secure database connectivity via **JDBC**.

### Features
- Add, update, delete, and view employee details  
- MySQL integration for data storage  
- GUI interface using Java Swing  
- Follows OOP principles and modular design  

### Tech Stack
Java | Swing | JDBC | MySQL | Eclipse
